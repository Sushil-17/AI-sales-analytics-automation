\# AI-Powered Sales Analytics Automation System



This project is an end-to-end data analytics automation pipeline that cleans messy enterprise sales data, validates business rules, generates KPI reports, loads data into PostgreSQL, and produces AI-generated business insights.



\## Key Features

\- Automated data ingestion from raw Excel files

\- Data cleaning: duplicate removal, null handling, datatype fixing

\- Business rule validation (Revenue = Quantity Ã— Unit Price)

\- Advanced KPI generation using pandas `.agg()`

\- Multi-sheet Excel KPI report generation

\- PostgreSQL integration for analytics-ready storage

\- Local AI-powered business insight generation using TinyLLaMA



\## Tech Stack

Python, pandas, PostgreSQL, SQLAlchemy, OpenPyXL, Ollama (TinyLLaMA)
